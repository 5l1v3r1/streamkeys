Thanks for using Streamkeys!
The default hotkeys are the media keys to control the players, and ALT+SHIFT+M to mute.
You can change these hotkeys to anything you like - on the chrome extension tab there is a link at the very bottom called "Keyboard shortcuts".

Supported media players:
  • 8track
  • Bankcamp
  • Deezer
  • Grooveshark
  • TheHypeMachine
  • MySpace
  • Pandora
  • Rdio
  • Spotify
  • Soundcloud
